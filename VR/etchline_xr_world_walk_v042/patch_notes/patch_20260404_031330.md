# Patch Notes - Etchline XR World Walk 0.4.0

- Stabilized floor calibration around a 6-foot-tall standing profile with a 1.70m eye line.
- Added tracking-space-aware floor settling, median head-height sampling, and a manual floor set action.
- Reworked XR anchor handling to stop head-height floor flips and reduce recenter jitter.
- Slowed ambient walkers and added interaction dialogue when you approach them.
- Added color customization for world and line palettes with automatic config saving.
- Added replaceable looping music placeholder in assets/music.
- Added robotic claw hands for controller-tracked XR presence, stars, and occasional overhead ships.
- Added a slow day/night hue cycle for the grid and structures with low-cost palette updates.
- Replaced the left wrist blaster with a long sword and upgraded the right-hand blaster silhouette.
- Tightened rendering cost with calmer palette updates, lighter sky effects, and fewer active transient systems.
- Extended the visual cycle so the grid, structure lines, and skyline drift through longer hue and saturation changes.
- Added a slow inversion pass that gradually pushes background and line colors toward opposite values across the day/night cycle.
